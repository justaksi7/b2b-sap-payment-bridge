SAP Payment Bridge Events Documentation

Servers

  • aws-sqs://sqs.eu-central-1.amazonaws.com/123456789012/EventDispatcherSQSaws-sqsEvent Dispatcher SQS

    AWS SQS Queue for storing succesful validations by sap that invokes the Event Dispatcher lambda after a specific batchSize or a batchWindow is reached. The VAlidator Lambda and the Worker Lambda publish messages to this Queue.

  • aws-sqs://sqs.eu-central-1.amazonaws.com/123456789012/POST-SQSaws-sqsPOST-SQS

    AWS SQS Queue for storing Payment POST Requests

  • aws-sqs://sqs.eu-central-1.amazonaws.com/123456789012/WORKER-SQSaws-sqsWorker-SQS

    AWS SQS Queue for storing PaymentCreated Events

  • aws-sqs://sqs.eu-central-1.amazonaws.com/123456789012/WEBHOOK-SQSaws-sqsWebhook-SQS

    AWS SQS Queue for storing PaymentCreated, PaymentSettled and PaymentFailed Events

  • aws-sqs://sqs.eu-central-1.amazonaws.com/123456789012/Logger-SQSaws-sqsLogger-SQS

    AWS SQS Queue for storing Logs

  • aws-lambda://lambda.eu-central-1.amazonaws.com/2015-03-31/functions/RequestDispatcher/aws-lambdaRequest Dispatcher

    AWS Lambda that receives HTTP Requests from the Gateway. If the Request is of type GET it returns the requested Payment information. If it is of type POST it writes into the POST-SQS and returns 202 Accepted with 'QUEUED'

  • aws-lambda://lambda.eu-central-1.amazonaws.com/2015-03-31/functions/WorkerLambda/invocationsaws-lambdaWorker Lambda

    AWS Lambda Worker Function that gets triggerd by the Worker-SQS and receives PaymentCreated events to transfer them to SAP. It changes the status of the Payment in the Payments Core database and creates either a PaymentSettled outbox event or PaymentFailed outbox event depending on the SAP response.

  • aws-lambda://lambda.eu-central-1.amazonaws.com/2015-03-31/functions/EventDispatcher/invocationsaws-lambdaEvent Dispatcher

    AWS Lambda Event Dispatcher function that queries the Payments Core database outbox event table and Partner Config database. It is invoked by the Event Dispatcher SQS and it dispatches the outbox events to the Worker-SQS and Webhook-SQS

  • aws-lambda://lambda.eu-central-1.amazonaws.com/2015-03-31/functions/WebhookLambda/invocationsaws-lambdaWebhook Lambda

    AWS Lambda Webhook function that is triggered by the Webhook-SQS it signs the messages from the queue and sends them to the partner specific webhook URLs.

  • aws-lambda://lambda.eu-central-1.amazonaws.com/2015-03-31/functions/LoggerLambda/invocationsaws-lambdaLogger Lambda

    AWS Lambda Logger that transfers the logs to Cloud Watch, Data Dog etc.

  • aws-lambda://lambda.eu-central-1.amazonaws.com/2015-03-31/functions/ValidationLambda/invocationsaws-lambdaValidation Lambda

    AWS Lambda Validator function that tries to ged a validation for partner payment requests from SAP. It is triggered by the POST-SQS. Depending ot the SAP response it either creates a payment and a PaymentCreated outbox event or only a PaymentFailed outbox event in to the Payments Core database and it also stores the partner configuration in the Partner Config database.

  • aws-lambda://lambda.eu-central-1.amazonaws.com/2015-03-31/functions/HealthCheck/invocationsaws-lambdaHealth Check Lambda

    AWS Lambda Health Check Function that is periodically by an EventBridge Scheduler. It checks the health status of SAP and sends the metric to CloudWatch

  • aws-lambda://lambda.eu-central-1.amazonaws.com/2015-03-31/functions/CircuitController/invocationsaws-lambdaCircuit Controller Lambda

    AWS Lambda Circuit Controller Function that reacts to alarms raised by CloudWatch and implements a Circuit Breaker Pattern by reducing, increasing or setting the SQS Consumer Concurrency to 0.

  • aws-rds://rds.eu-central-1.amazonaws.com/123456789012/payments-core-dbaws-rdsPayments Core

    AWS RDS Payments Core Database with 2 tables - payments and outbox events.

  • aws-rds://rds.eu-central-1.amazonaws.com/123456789012/partner-config-dbaws-rdsPartner Config

    AWS RDS Partner Configuration Database it stores partner configurations such as mapping rules, webhook URLs, webhook secrets etc.

  • aws-eventbridge://aws.events.eu-central-1/123456789012:schedule/dispatcher-pollaws-eventbridgeeventbridge-scheduler

    AWS EventBridge Scheduler, konfiguriert mit rate(30 seconds), triggert Dispatcher Lambda.

  • aws-sqs://rds.eu-central-1.amazonaws.com/123456789012/dlq-exampleaws-sqsDLQ-Example

    An example to showcase the transfer of events with status 'FAILED' to a Dead Letter Queue.
